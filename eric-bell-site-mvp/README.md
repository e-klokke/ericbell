# Eric Bell Personal Brand Site MVP

This package includes two versions of the MVP website concept.

## 1. HTML Prototype
Path: `html-prototype/index.html`

Open this file directly in your browser to preview the site immediately.

## 2. Next.js App
Path: `nextjs-app/`

Use this version for a production-ready build with Codex, Vercel, or a developer.

### Run locally
```bash
cd nextjs-app
npm install
npm run dev
```

Then open the local URL shown in your terminal.
